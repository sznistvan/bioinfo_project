    About topic

    This is how to use it:

  • Run: python translator.py "TCAATCGTGGCCCGA"
  • Executable: translator.exe

    • Rule 1

    DNA sequence should contain only valid bases! Bases: adenine (A), guanine (G), cytosine (C), and thymine (T)

    Rule 2

    The DNA sequence's lenght should be devided by 3, since the ribosome is picky. :(
